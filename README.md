#  Tourism Website – Travel Mania

A static tourism website designed to provide users with information about travel destinations, visa guidelines, activities, and galleries.

##  Features

* Multi-page website with structured navigation
* Dedicated pages for:

  * Home
  * Destinations
  * Visa Information
  * Things to Do
  * Gallery
  * Contact
* Responsive layout using CSS
* Clean and simple UI design

##  Technologies Used

* HTML5
* CSS3
* Notepad++ (for development)

##  Project Structure

* `index.html` – Homepage
* `destination.html` – Travel destinations
* `visa.html` – Visa-related information
* `thingstodo.html` – Activities and attractions
* `gallery.html` – Image gallery
* `contact.html` – Contact page
* `style_webpage.css` – Styling

##  How to Run

1. Download or clone the repository
2. Open `index.html` in any web browser

##  Note

This project was developed as part of an undergraduate coursework and is not deployed online.
